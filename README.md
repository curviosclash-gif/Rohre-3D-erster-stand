# CuviosClash

CuviosClash ist ein 3D-Arcade-Flugspiel auf Basis von Three.js.
Dieses Repository enthaelt Runtime, Editor, Tests und Bot-Training.

## Schnellstart

```bash
npm install
npm run dev
```

## Wichtige Befehle

```bash
npm run build
npm run dev:logged
npm run logs:clean
npm run test:core
npm run docs:sync
npm run docs:check
```
