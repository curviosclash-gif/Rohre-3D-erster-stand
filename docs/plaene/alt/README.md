# Alte Plaene

Hier liegen uebernommene oder abgeloeste Implementierungsplaene.

Stand: 2026-04-01

- Uebernommene externe Plaene aus `docs/plaene/neu/` werden nach dem Intake hierher verschoben.
- Aktive kanonische Blockdetails liegen nicht hier, sondern unter `docs/plaene/aktiv/`.
- Abgeschlossene Blockarchive aus `docs/Umsetzungsplan.md` liegen ebenfalls hier, soweit sie blockweise konsolidiert vorliegen.
- Historische komplette Masterplan-Snapshots bleiben weiterhin unter `docs/archive/plans/superseded/`.
