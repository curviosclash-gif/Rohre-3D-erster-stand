description: Turn a rough idea into a scoped, evaluable concept.
---

## 0. Capture

- What is the idea? What player value? Any reference examples?

## 1. Create concept doc

Create `docs/Ideen/[Name].md`:
- Description, gameplay impact, scope/affected modules
- Variants (minimal, standard, full)
- Open questions, score (fun, feasibility, fit)

## 2. Decision

- Discard / Backlog / Plan for implementation

## 3. Next action

- If approved: start `/plan`.
